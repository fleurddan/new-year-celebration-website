A minimal interactive website created to deliver messages through a sequenced digital experience. This project displays animated messages one by one, accompanied by subtle visuals and background music.

* **Minimal Design:** A clean and focused single-page interface.
* **Background Music:** Audio integration optimized for user interaction.
* **Responsive Layout:** Fully compatible with mobile and desktop devices.

## Tech Stack
* **HTML**
* **CSS**
* **JavaScript**

## Implementation
The project is designed to transform a standard text greeting into a structured digital moment. It focuses on the timing and presentation of content to enhance the delivery of the message.

1. Clone the repository.
2. Update the message array in the script file.
3. Replace the audio file in the assets directory.
4. Deploy via GitHub Pages.
